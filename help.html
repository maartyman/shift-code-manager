üìä Understanding Code States

Code Overview
The Code Overview section shows different states:
  • New: Fresh codes ready to be redeemed
  • Redeemed: Successfully redeemed codes
  • Validated: Codes confirmed as already redeemed
  • Expired: Codes that are no longer valid
  • Errors: Codes that encountered issues

‚öôÔ∏è Settings Explained

Timing Settings

  • Code Delay: Time to wait between redeeming individual codes. The SHIFT website rate limits requests, so we wait a few seconds between each code redemption to avoid being blocked. 6 seconds is recommended.
  • Retry Delay: Time to wait before retrying failed redemptions

Code Sources

  • Pre-configured reliable websites that publish SHIFT codes
  • Add custom URLs if you know other good sources
  • Different games can have different sources
  • Important: To update sources for a specific game, first select that game in the Main tab, then go to Settings

Settings Backup

  • Export: Download your settings and code history
  • Import: Restore settings from a backup file

üîß Troubleshooting

Codes Not Redeeming?
  • Make sure you're logged into shift.gearboxsoftware.com
  • Check if codes are already redeemed or expired
  • Try switching platforms (Steam, Epic, etc.)
  • Disable other browser extensions temporarily
  • Clear browser cache and cookies for SHIFT website
Notifications Not Working?
  • Check browser notification permissions
  • Ensure notifications are enabled in extension settings
  • Try shorter check intervals for testing
  • Verify notification settings in browser preferences
Extension Not Loading?
  • Refresh the browser page
  • Check for browser updates
  • Try disabling other extensions
  • Reinstall the extension if necessary

‚ö†Ô∏è Important Notes

Rate Limiting
The extension automatically waits between code redemptions because the SHIFT website will stop accepting codes if they're submitted too quickly. A 6-second delay between codes is recommended to avoid being temporarily blocked.

Login Required
You must be logged into your SHIFT account on shift.gearboxsoftware.com before using the redemption feature. The extension cannot log you in automatically.
Privacy & Security
  • All data stored locally in your browser
  • No external servers or data collection
  • Open source code - audit it yourself
  • Only requests necessary permissions
