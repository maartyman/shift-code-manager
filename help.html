üîß Troubleshooting

Common Issues
  • Login Required: If redemption fails, check that you are still logged into the SHIFT website.
  • Rate Limiting: Use the "Code Delay" setting (default 5s) to prevent blocking.
  • Missing Elements: If the extension looks wrong, try reloading the extension or clearing browser cache.
Important Note
This extension runs locally on your machine. No data is sent to external servers other than the configured code sources and the official SHIFT website.
